模拟实际项目开发流程，依次进行如下操作，所有操作均在IDEA下完成：
1. 项目拉取(clone)
2. 切换分支(checkout)
3. 更新(pull)
4. 修改代码
5. 提交(commit)
6. 追加提交(amend commit)
7. 上传远程仓库(push)
8. 解决冲突(merge or rebase)
9. 再次上传
